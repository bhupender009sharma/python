multiplayer rock paper scissor game that can be played over a
connected network

-> can simultaneously hold multiple games in pairs of two

-> uses python 'socket' module for communicating 

-> uses concepts of 'threading'

-> uses python 'pickle' module for sending and receiving objects 


-> for running game ->

		-> change the server variable value from the 
		   server.py file to the ip address over which 
		   you are connected
		
		->first run server.py file

		-> then run client file in pairs of two for 
		   creating multiple games
